- src/camera-follow.ts
	- A camera behavior that smoothly follows the local player's position using linear interpolation, with a configurable smooth factor for adjusting the follow speed. The behavior runs only on the client side and uses the @syncedValue decorator to network the smooth factor property, making it adjustable during runtime. It depends on a PlayerSpawner class to obtain the local player reference and updates the camera entity's position every post-tick.
- src/moving-platform.ts
	- A moving platform behavior that creates automated platform movement along predefined paths, supporting both linear and L-shaped trajectories with configurable movement types (horizontal/vertical), directions, and speeds. The platform's movement is synchronized across the network using @syncedValue decorators, and it implements smooth interpolation between waypoints while maintaining authority on the server side. The behavior includes extensive configuration options for path shapes, initial directions, and movement constraints, making it suitable for creating various types of moving platforms in platformer-style games.
- src/platform-player.ts
	- A platform game character controller that implements physics-based movement, jumping mechanics with coyote time, and special handling for moving platforms. The behavior manages player states including grounded detection, jump acceleration, and platform sticking, while using ray casting for precise collision detection with platforms and ceilings. Features a complete animation system that transitions between idle, run, jump, and fall states based on the character's movement state.
- src/player-spawner.ts
	- PlayerSpawner handles the creation of player entities in a multiplayer game environment, specifically on the client side. It uses synchronized values to reference a player prefab, spawn point, and container entity, then instantiates a new player instance at the specified spawn point with proper network authority settings, storing the local player reference in a static variable for global access.
