Rebur Engine Documentation

Rebur is a multiplayer-first 2D game engine built on TypeScript. Games run simultaneously for all players, with shared state synchronized automatically over WebSockets.

💡 New here? Open the editor via the launcher, pick a world like Tutorial or Template Project, click Edit, and follow along with the examples below.

Core Concepts

  • Entities — everything in a game world (players, platforms, bullets, cameras). Entities form a hierarchy (parent → children).
  • Behaviors — TypeScript scripts attached to entities. They hold logic and state.
  • Components — built-in engine pieces like Sprite2D, RigidBody2D, Camera2D. Attached to entities alongside behaviors.
  • Worlds — a project folder containing a project.json scene and a src/ directory of behavior scripts.

Quick Start — Your First Behavior

Open any world in the editor, right-click the scene graph → Add Entity. Then in the Inspector, click + Add Behavior and choose New Script. Give it a name — the editor creates a file in src/.

Here is the simplest possible behavior — moves an entity to the right every frame:

import { Behavior } from "@dreamlab/engine";

export default class MoveRight extends Behavior {
  onTick(): void {
    this.entity.transform.position.x += 2;
  }
}

Save the file (Ctrl+S), hit the ▶ Play button, and watch it move.

Adding Speed as an Inspector Property

import { Behavior } from "@dreamlab/engine";

export default class MoveRight extends Behavior {
  // This property shows up in the Inspector panel on the right
  speed = this.define("speed", 2);

  onTick(): void {
    this.entity.transform.position.x += this.speed.value;
  }
}
📝 Note Import paths use @dreamlab/engine — this is the internal package name. All examples below use this import path.

Behaviors

A Behavior is a TypeScript class that extends Behavior from @dreamlab/engine. Every script file should have a single default export of a Behavior class.

import { Behavior, Vector2 } from "@dreamlab/engine";

export default class MyBehavior extends Behavior {
  // Inspectable property (number)
  speed = this.define("speed", 5);

  // Inspectable string property
  label = this.define("label", "Hello");

  onSpawn(): void {
    console.log("Entity spawned!", this.entity.name);
  }

  onTick(): void {
    // Runs every frame (~60fps)
  }

  onDestroy(): void {
    // Cleanup when entity is removed
  }
}

Accessing Other Behaviors

const rb = this.entity.getBehavior(RigidBody2D);
const health = this.entity.getBehavior(HealthBehavior);
if (health) health.takeDamage(10);

Accessing Child Entities

// By name
const child = this.entity.children.get("BulletSpawn");

// Iterate all children
for (const child of this.entity.children) {
  console.log(child.name);
}

// Parent
const parent = this.entity.parent;

Lifecycle Methods

MethodWhen it runs
onSpawn()Once when the entity enters the world (on spawn or scene load).
onTick()Every frame. Use for movement, input checks, AI, etc.
onDestroy()When the entity is removed. Clean up subscriptions and timers here.
onCollide(other)When this entity's collider touches another (requires RigidBody2D).

Timer / Delay

import { Behavior } from "@dreamlab/engine";

export default class DelayExample extends Behavior {
  private timer = 0;
  interval = this.define("interval", 2); // seconds

  onTick(): void {
    this.timer += this.game.time.delta; // delta in seconds
    if (this.timer >= this.interval.value) {
      this.timer = 0;
      console.log("Ping!");
    }
  }
}

Use this.define(name, defaultValue) to create properties that appear in the Inspector and are saved in the scene file.

// later:

Input Handling

Access input through this.game.input:

import { Behavior, Vector2 } from "@dreamlab/engine";

export default class PlayerInput extends Behavior {
  speed = this.define("speed", 4);

  onTick(): void {
    const input = this.game.input;
    const move = new Vector2(0, 0);

    if (input.isKeyDown("ArrowLeft")  || input.isKeyDown("a")) move.x -= 1;
    if (input.isKeyDown("ArrowRight") || input.isKeyDown("d")) move.x += 1;
    if (input.isKeyDown("ArrowUp")   || input.isKeyDown("w")) move.y -= 1;
    if (input.isKeyDown("ArrowDown") || input.isKeyDown("s")) move.y += 1;

    if (move.magnitude() > 0) move.normalize();
    this.entity.transform.position.x += move.x * this.speed.value;
    this.entity.transform.position.y += move.y * this.speed.value;

    if (input.isKeyPressed(" ")) {
      // Space was just pressed this frame (not held)
      console.log("Jump!");
    }
  }
}

Input Methods

MethodReturnsDescription
input.isKeyDown(key)booleanTrue every frame the key is held.
input.isKeyPressed(key)booleanTrue only on the first frame the key goes down.
input.isKeyReleased(key)booleanTrue only on the frame the key is released.
input.cursorVector2Mouse/pointer position in world space.
input.isMouseDown(btn)booleanTrue while mouse button (0=left, 2=right) is held.
📱 Mobile On mobile, the virtual joystick and action buttons fire the same keyboard events, so the same input code works on touch devices without any changes.

Transform & Movement

const t = this.entity.transform;

// Position (world space, pixels)
t.position.x = 100;
t.position.y = 200;
t.position = { x: 0, y: 0 };

// Rotation (degrees)
t.rotation += 1; // rotate 1° per frame

// Scale
t.scale.x = 2; // double width
t.scale.y = 2; // double height

// Local vs World (for parented entities)
// t.position is always world-space
// t.localPosition is relative to parent

Smooth Movement with Delta Time

onTick(): void {
  const dt = this.game.time.delta; // seconds since last frame
  const spd = 200; // pixels per second
  this.entity.transform.position.x += spd * dt;
}

Physics (Rapier2D)

Add a RigidBody2D component to an entity to enable physics. The engine uses Rapier2D for fast, deterministic simulation.

import { Behavior, RigidBody2D } from "@dreamlab/engine";

export default class PhysicsPlayer extends Behavior {
  private rb!: RigidBody2D;

  onSpawn(): void {
    this.rb = this.entity.getBehavior(RigidBody2D)!;
  }

  onTick(): void {
    const input = this.game.input;

    // Apply horizontal force
    if (input.isKeyDown("ArrowLeft") || input.isKeyDown("a")) {
      this.rb.applyImpulse({ x: -5, y: 0 });
    }
    if (input.isKeyDown("ArrowRight") || input.isKeyDown("d")) {
      this.rb.applyImpulse({ x: 5, y: 0 });
    }

    // Jump (check if grounded first)
    if (input.isKeyPressed(" ")) {
      this.rb.applyImpulse({ x: 0, y: -300 });
    }
  }
}

Physics Properties

Property / MethodDescription
rb.velocityCurrent velocity Vector2.
rb.applyImpulse(v)Apply an instant impulse force.
rb.applyForce(v)Apply a continuous force (use in onTick).
rb.isGroundedTrue when touching a surface below.
rb.gravityScaleMultiplier for gravity (0 = no gravity).
rb.lockRotationPrevents physics rotation.

Entity System

Spawning Entities at Runtime

import { Behavior } from "@dreamlab/engine";

export default class Spawner extends Behavior {
  onTick(): void {
    if (this.game.input.isKeyPressed("e")) {
      const bullet = this.game.world.spawn({
        type: Entity,
        name: "Bullet",
        transform: {
          position: { x: this.entity.transform.position.x, y: this.entity.transform.position.y },
          scale: { x: 1, y: 1 },
        },
        behaviors: [{ type: BulletBehavior }],
      });
    }
  }
}

// Destroy self
this.entity.destroy();

// Destroy another entity
other.destroy();

// Find by name in world

Sprite Example

import { Behavior, Sprite2D } from "@dreamlab/engine";

export default class AnimateColor extends Behavior {
  private sprite!: Sprite2D;

  onSpawn(): void {
    this.sprite = this.entity.getBehavior(Sprite2D)!;
    this.sprite.texture = "res://assets/player.png";
  }

  onTick(): void {
    // Tint based on time
    const t = this.game.time.now;
    this.sprite.tint = Math.sin(t) > 0 ? 0xff0000 : 0xffffff;
  }
}

Multiplayer & Networking

Rebur is multiplayer-first. Any entity spawned on the server automatically replicates to all clients. Behavior properties defined with this.define() and decorated with @synchronized sync automatically.

Synchronized Properties

import { Behavior, synchronized } from "@dreamlab/engine";

export default class Health extends Behavior {
  // This value is automatically kept in sync across all players
  @synchronized
  hp = this.define("hp", 100);

  takeDamage(amount: number): void {
    this.hp.value -= amount; // Change replicates to all clients
  }
}

Client vs Server Logic

onTick(): void {
  if (this.game.isServer) {
    // Only runs on the server (authoritative)
    this.updateAI();
  }

  if (this.game.isClient) {
    // Only runs on the client (visual/input)
    this.handleInput();
  }
}

Remote Procedure Calls (RPC)

import { Behavior, rpc } from "@dreamlab/engine";

export default class Shooter extends Behavior {
  // Client calls this → server executes it
  @rpc({ from: "client", to: "server" })
  requestFire(direction: number): void {
    // Server-side: spawn bullet, validate, etc.
    this.spawnBullet(direction);
  }
}

UI / HUD

Rebur supports React-based UI via the @dreamlab/ui package. UI renders on top of the game canvas and receives input independently.

import { Behavior } from "@dreamlab/engine";
import { createElement } from "@dreamlab/ui";

export default class HUD extends Behavior {
  onSpawn(): void {
    this.game.ui.render(() => (
      <div style={{ position: 'absolute', top: 16, left: 16, color: 'white', fontSize: 20 }}>
        HP: {this.hp}
      </div>
    ));
  }
}
